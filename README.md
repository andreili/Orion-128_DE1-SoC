# Орион-128

Реализация бытового компьютера "Орион-128" на FPGA. Проект предназначен для платы Terasic DE1-SoC.
Реализовано в схемном варианте (за исключением портов ввода-вывода), со следующими расхождениями:
- Сканирование видеопамяти и вывод видеосигналов адаптированы для VGA-мониторов;
- Для разрешения проблем с арбитражем для видеопамяти используется двухпортовая статическая память, дополнителоьные банки памяти - обычная статическая память;
- Добавлено динамическое переключение режимов вывода изображения переключателем SW9. Нижнее положение соответствует стандартному режиму 384 точки, верхнее - расширенному режиму 480 точек;
- Добавлено динамическое переключение разрешения изображения - 640х480 или 640х350 переключателем SW8;
- В качестве процессора используется модель, основанная на реверс-инжиниринге оригинального процессора КР580ВМ80А (https://github.com/1801BM1/vm80a).

Ход разработки обсуждается на форуме - http://zx-pk.ru/threads/28763-eshche-odin-orion-na-plis.html

# Статус проекта
- Реализована базовая версия компьютера, корректность работы проверена тестами ОЗУ (TEST128, TEST256);
- Первичная реализация Z80-Card II, пока что с ошибками в арбитраже памяти;
- Не реализовано расширение памяти до 512Кбайт - внутрення память не может иметь более чем 64К слов.

# Подготовка и сборка
Для облегчения проекта некоторые файлы не были включены в репозиторий, поэтому для сборки необходимо проинициализировать IP-модули PLL. Для этого достаточно в дереве проекта дважды кликнуть по ним и в появившемся окне мастера конфигураций нажать "Finish".
Для конвертации ROM-образов в формат, принимаемый Qurtus'ом, необходимо:
- Любой удобной программой преобразовать двоичный файл в формат Intel HEX;
- Открыть данный файл Quartus'ом ("File" -> "Open..." -> "All files (*.*)");
- В диалоговом окне параметр "Word size" оставить без изменений (8) и нажать "OK";
- В главном меню выбрать "File" -> "Save as...", выбрать место для сохранения.

**Внимание! Сохранённый HEX-файл будет добавлен в текущий проект!**
